Section 01

Product Vision

AI-native business automation — like Legacy SOAR, but for every team, every process, every industry.

What It Is

Lumen Flow is an intelligent workflow automation platform. Deterministic flows handle the predictable parts. AI nodes handle the uncertain parts. Agent teams handle the complex parts. Together, they automate any repeatable business process — from customer support to legal review to DevOps incident response.

Positioning: "Intelligent workflow automation for operations teams"

Three-Layer Moat

  • AI Flow Generation — LLMs generate complete flows from natural language descriptions. No competitor has this.
  • Agent Team Nodes — Flow nodes can spawn multi-agent AI teams via agent-leader integration.
  • Lumen Ecosystem — Memory, behavioral profiles, Telegram, MCP servers — a connected intelligence layer.

Target Market

  • Operations teams managing repeatable processes
  • Customer success teams with case-based SLA work
  • IT teams orchestrating multi-system workflows
  • Any team that needs deterministic flows + AI intelligence

Why Now

LLMs are finally good enough to generate structured workflows. MCP creates a universal connector protocol. Agent orchestration is maturing. The convergence of these three technologies makes AI-native workflow automation possible for the first time.

Section 05

Key Differentiators vs Legacy SOAR

Lumen Flow is inspired by Legacy SOAR but designed without its limitations. Here are the four major advantages.

Multi-Flow per Case

Legacy SOAR
1 playbook per incident. To run multiple processes, you need separate incidents or complex sub-playbooks.
vs
Lumen Flow
A case can trigger multiple flows simultaneously or sequentially. Case tracks all flows' combined state.

Universal Triggers

Legacy SOAR
Incident-triggered only. Playbooks start when a security incident is ingested from an integration.
vs
Lumen Flow
10+ trigger types: webhook, cron, email, event bus, API, manual, chat command, data feed, entity change, flow chaining.

AI Flow Generation

Legacy SOAR
Manual-only playbook creation. Drag-and-drop in the visual editor or write YAML.
vs
Lumen Flow
Describe in natural language → LLM generates flow. Three modes: generate, modify, explain. Via chat or prompt studio.

Agent Team Nodes

Legacy SOAR
Single-script task execution. Each automation step runs one Python/JS script in isolation.
vs
Lumen Flow
Spawn multi-agent AI teams via agent-leader. Agents collaborate, delegate, review each other's work.

Section 09

Lumen Ecosystem Integration

Lumen Flow is the automation layer in a broader intelligence ecosystem. Flows connect to agents, memory, messaging, and external services through MCP.

The Meta-Orchestrator Pattern

Lumen Flow drives agent-leader teams from within flow nodes. When a flow node requires complex AI work, it spawns an agent team, polls for progress, surfaces blockers, and collects results on completion. This creates a meta-orchestration layer where deterministic flows control non-deterministic AI agents.

Flow Node (Agent Team) → orchestrator_headless_start()
  → Poll: orchestrator_headless_status() every 30s
  → Surface blockers: orchestrator_list_blockers()
  → On completion: collect results, continue flow
  → On failure: trigger fallback branch in flow
